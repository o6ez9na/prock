# 🚀 PRock

**PRock** — современная веб-панель для управления прокси и VPN-аккаунтами.
Позволяет создавать пользователей, выдавать прокси-конфигурации и контролировать доступ через удобный интерфейс 🌐.

---

## ✨ Основные возможности

- 👤 Управление пользователями и группами  
- 🔑 Создание и выдача прокси/VPN аккаунтов  
- 🌐 Поддержка нескольких типов прокси и протоколов  
- 📊 Статистика использования и логирование подключений  
- 💻 Интуитивно понятный веб-интерфейс  
- 🔗 Возможность интеграции с внешними сервисами

---

## 🛠 Технологии

- **Frontend:** React, Tailwind CSS / Chakra UI  
- **Backend:** FastAPI / Python  
- **База данных:** PostgreSQL  
- **Docker:** для быстрого развертывания всей инфраструктуры  
- **Git Submodules:** для разделения frontend/backend
